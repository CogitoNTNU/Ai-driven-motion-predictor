    src.Kaare.models

    Domain models for the Kaare data layer.

    Classes:

    • ArticleSentiment –

      FinBERT sentiment scores for a single article.

    • DailyMarketSentiment –

      Aggregated market-wide sentiment for a single trading day.

    • DailyTickerSentiment –

      Aggregated per-ticker sentiment for a single trading day.

    • MacroData –

      Macro-economic data for a single date.

    • NewsSentimentResult –

      FinBERT sentiment result for a stock ticker over a date range.

    • RawNews –

      A single raw news article from the HuggingFace dataset.

    • StockOHLCV –

      OHLCV record for a single stock on a single date.

    ArticleSentiment dataclass 

    ArticleSentiment(raw_news_id: int, positive: float, negative: float, neutral: float, net_score: float)

    FinBERT sentiment scores for a single article.

    Attributes:

    • raw_news_id (int) –

      FK to raw_news.id.

    • positive (float) –

      Probability of positive sentiment.

    • negative (float) –

      Probability of negative sentiment.

    • neutral (float) –

      Probability of neutral sentiment.

    • net_score (float) –

      positive - negative, range [-1, 1].

    DailyMarketSentiment dataclass 

    DailyMarketSentiment(trading_date: date, mean_score: float, median_score: float, article_count: int, std_score: float | None = None, pct_positive: float | None = None, pct_negative: float | None = None, mean_score_3d: float | None = None, mean_score_7d: float | None = None)

    Aggregated market-wide sentiment for a single trading day.

    Attributes:

    • trading_date (date) –

      The trading day.

    • mean_score (float) –

      Average net sentiment across all articles.

    • median_score (float) –

      Median net sentiment.

    • article_count (int) –

      Number of articles scored.

    • std_score (float | None) –

      Standard deviation of net scores.

    • pct_positive (float | None) –

      Fraction of articles with positive probability > 0.5.

    • pct_negative (float | None) –

      Fraction of articles with negative probability > 0.5.

    • mean_score_3d (float | None) –

      3-day rolling average of mean_score.

    • mean_score_7d (float | None) –

      7-day rolling average of mean_score.

    DailyTickerSentiment(trading_date: date, ticker: str, mean_score: float, article_count: int, std_score: float | None = None, pct_positive: float | None = None, pct_negative: float | None = None)

    Aggregated per-ticker sentiment for a single trading day.

    Attributes:

    • trading_date (date) –

      The trading day.

    • ticker (str) –

      Ticker symbol.

    • mean_score (float) –

      Average net sentiment for this ticker.

    • article_count (int) –

      Number of articles tagged with this ticker.

    • std_score (float | None) –

      Standard deviation of net scores.

    • pct_positive (float | None) –

      Fraction of articles with positive probability > 0.5.

    • pct_negative (float | None) –

      Fraction of articles with negative probability > 0.5.

    MacroData dataclass 

    MacroData(date: date, gold_price: float | None, treasury_yield_10y: float | None, vix: float | None = None, news_sentiment_score: float | None = None)

    Macro-economic data for a single date.

    Attributes:

    • date (date) –

      Date of the record.

    • gold_price (float | None) –

      Spot gold price in USD, or None if unavailable.

    • treasury_yield_10y (float | None) –

      10-year US Treasury yield (%), or None if unavailable.

    • news_sentiment_score (float | None) –

      Daily market news sentiment in [-1.0, 1.0] computed by FinBERT, or None if not yet computed.

    NewsSentimentResult dataclass 

    NewsSentimentResult(symbol: str, start: date, end: date, article_count: int, avg_score: float, daily_scores: dict[date, float] = dict())

    FinBERT sentiment result for a stock ticker over a date range.

    Attributes:

    • symbol (str) –

      Ticker symbol that was analysed.

    • start (date) –

      Inclusive start date of the news window.

    • end (date) –

      Inclusive end date of the news window.

    • article_count (int) –

      Number of articles scored.

    • avg_score (float) –

      Average FinBERT net sentiment (positive - negative) in [-1.0, 1.0].

    • daily_scores (dict[date, float]) –

      Per-date average sentiment scores.

    RawNews dataclass 

    RawNews(date_utc: datetime, trading_date: date, text: str, text_hash: str, dataset_subset: str, source: str | None, tickers: list[str] = list(), id: int | None = None)

    A single raw news article from the HuggingFace dataset.

    Attributes:

    • date_utc (datetime) –

      Publication timestamp in UTC.

    • trading_date (date) –

      NYSE-aligned trading date.

    • text (str) –

      Article text (headline + body).

    • text_hash (str) –

      SHA-256 of text for deduplication.

    • dataset_subset (str) –

      Source subset name (e.g. "fnspid_news").

    • source (str | None) –

      Publisher name, or None if unknown.

    • tickers (list[str]) –

      List of associated ticker symbols.

    • id (int | None) –

      Database-assigned ID (None before insertion).
